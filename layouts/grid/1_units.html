We can define the width of rows or columns as follows,

  1. Individually: grid-template-*: 150px 150px 200px; Used to define different values
  2. Using repeat fn: grid-template-*: repeat(2, 150px) 200px; Used to avoid repeatability
  3. Using fractional-unit(fr): grid-template-*: repeat(2, 1fr) 200px; Used to equally divide available space values to specified rows/columns. available space(gap value - is not taken into account)
  4. Other units: %, px, rem, em. Other units are also accepted
